A console-based C++ application built to practice core programming fundamentals such as functions, arrays, control flow, and algorithmic reasoning.
-------------------------------------------

**Features**

Calculate sum of array elements

Calculate average

Find median (uses bubble sort)

Find second largest value

Count occurrences of a specific number

Interactive menu using do-while and switch

-------------------------------------------
***Concepts Practiced***

Functions with return values

Pass-by-value vs pass-by-reference

Nested loops

Bubble sort algorithm

In-place array manipulation

Input validation

Control flow (switch, do-while)

Basic algorithm design
