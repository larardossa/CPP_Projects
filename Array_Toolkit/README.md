# Array Toolkit

A console-based C++ utility for practicing core array algorithms.

## Features
- Remove duplicates (unsorted)
- Swap adjacent elements
- Reverse array
- Push zeros to the back
- Bubble sort
- Find min and max

## Concepts Used
- Nested loops
- do-while and switch
- In-place modification
- Pass-by-reference
- Algorithmic thinking
